# Dytallix Explorer

Block explorer for Dytallix — displays blocks, transactions, addresses, validators, epoch data, governance proposals, and the dimensional fee breakdown showing C-Gas and B-Gas separately.

## Status

Under construction. Live explorer at https://explorer.dytallix.com

## What It Shows

- Blocks with proposer, slot, epoch, C-Gas used, B-Gas used
- Transactions with hash, status, fee breakdown
- Addresses with DGT and DRT balances
- Validators with stake weight, uptime, slash history
- Epoch data and VRF entropy
- Governance proposals and vote counts
- Dimensional fee market: C-Gas and B-Gas displayed separately
